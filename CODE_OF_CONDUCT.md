# Code of Conduct

## Our standard

We want `welight-cli` to be a respectful, constructive, and welcoming project.

Examples of positive behavior:

- Being respectful in language and tone
- Giving actionable, specific feedback
- Assuming good intent while discussing tradeoffs
- Keeping the focus on improving the product and helping other contributors

Examples of unacceptable behavior:

- Harassment, threats, or personal attacks
- Discriminatory or hateful language
- Trolling, deliberate disruption, or bad-faith argument
- Publishing private information without permission

## Scope

This code of conduct applies in:

- GitHub issues
- Pull requests
- Discussions related to this project

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, commits, or pull requests that violate this code of conduct.

If you experience or witness unacceptable behavior, report it privately through the contact path described in [SECURITY.md](./SECURITY.md) or through the maintainer contact information on [xuyi.dev](https://xuyi.dev).
