# 指令帮助文档

这份文档用于集中维护 `welight-cli` 的命令说明、用途和常见示例。

如果命令、参数、工作流或推荐用法发生变化，请同步更新本文件。

## 全局入口

### `wl`

用途：

- 启动 Welight CLI
- 首次运行时自动进入 setup 引导

示例：

```bash
wl
wl --help
wl --version
```

## Setup

### `wl setup`

用途：

- 首次初始化 CLI
- 重新进入配置向导

说明：

- 会引导配置许可证、AI 文本模型、Tavily 实时网络搜索、AI 图片模型、微信公众号信息

示例：

```bash
wl setup
wl setup --section license
wl setup --section ai
wl setup --section wechat
```

## Auth

### `wl auth login`

用途：

- 保存许可证信息并验证当前设备状态

示例：

```bash
wl auth login
wl auth login --licenseKey 'YOUR-LICENSE' --customerEmail 'you@example.com'
```

### `wl auth status`

用途：

- 查看当前保存的许可证状态

示例：

```bash
wl auth status
wl auth status --json
```

### `wl auth logout`

用途：

- 清除本地保存的许可证信息

示例：

```bash
wl auth logout
```

## Config

### `wl config path`

用途：

- 查看本地配置目录、配置文件路径和许可证文件路径

示例：

```bash
wl config path
wl config path --json
```

### `wl config list`

用途：

- 列出当前 CLI 配置
- 默认会对密钥和密文类字段做脱敏显示

示例：

```bash
wl config list
wl config list --show-secrets
wl config list --json
```

### `wl config get <key>`

用途：

- 读取单个配置项

示例：

```bash
wl config get ai.defaultModel
wl config get wechat.proxyOrigin
wl config get search.provider
```

### `wl config set <key> <value>`

用途：

- 写入单个配置项

示例：

```bash
wl config set ai.defaultModel glm-46
wl config set wechat.proxyOrigin https://waer.ltd
wl config set search.provider tavily
```

### `wl config unset <key>`

用途：

- 删除单个配置项

示例：

```bash
wl config unset ai.image.endpoint
```

### `wl config export`

用途：

- 导出当前应用配置

示例：

```bash
wl config export
wl config export -o ./wl-config.json
```

### `wl config import <file>`

用途：

- 从 JSON 文件导入应用配置

示例：

```bash
wl config import ./wl-config.json
wl config import ./wl-config.json --merge
```

## AI

### `wl ai create`

用途：

- 基于一个主题或提示词生成公众号文章 Markdown
- 默认会先走 Tavily 实时搜索，再生成文章内容
- 如果你明确不希望联网搜索，可使用 `--no-webSearch`

示例：

```bash
wl ai create --prompt "Write a WeChat article about AI workflows"
wl ai create --prompt "Write a product postmortem" --output ./article.md
wl ai create --prompt "Write a WeChat article based on the latest AI news"
wl ai create --prompt "Write a timeless article about note-taking" --no-webSearch
```

### `wl ai title`

用途：

- 基于文章内容生成 5-10 个标题推荐
- 自动解析标题评分、推荐星级和分析说明
- 可选联动生成封面搜索词并拉取封面推荐

示例：

```bash
wl ai title ./article.md
wl ai title --text "Paste the full article body here"
wl ai title ./article.md --output ./title-recommendations.md
wl ai title ./article.md --no-withCover
```

## Article

### `wl article compose`

用途：

- 走完整的文章创作工作流
- 支持输入内容、AI 创作或整理、排版、选主题、导出、复制或发布
- 当工作流中使用 AI 创作文章时，默认会先走 Tavily 实时搜索
- 如果你明确不希望联网搜索，可使用 `--no-webSearch`

示例：

```bash
wl article compose
wl article compose --input ./article.md
wl article compose --url https://example.com/post
wl article compose --prompt "Write a WeChat article about AI products"
wl article compose --prompt "Write about the latest game industry news"
wl article compose --prompt "Write an evergreen article about design systems" --no-webSearch
```

## Cover

### `wl cover check <file>`

用途：

- 检查文章是否具备可用封面

示例：

```bash
wl cover check ./article.html
wl cover check ./article.md --coverImage ./cover.png
```

### `wl cover generate`

用途：

- 根据标题和摘要生成封面图

示例：

```bash
wl cover generate --title "AI Workflow Trends"
wl cover generate --title "AI Workflow Trends" --summary "A WeChat article about AI workflow changes"
```

### `wl cover recommend`

用途：

- 基于文章内容或直接输入的查询词推荐一张更适合作为封面的图片
- 默认使用 Pixabay 搜索并自动挑选更适合的结果

示例：

```bash
wl cover recommend ./article.md
wl cover recommend --query "ai workflow laptop dashboard"
wl cover recommend --text "Paste the article content here"
```

## Copy

### `wl copy wechat <file>`

用途：

- 把公众号兼容内容写入系统剪贴板

示例：

```bash
wl copy wechat ./article.html
```

## Publish

### `wl publish wechat <file>`

用途：

- 把文章推送到公众号草稿箱或正式发布

示例：

```bash
wl publish wechat ./article.html --mode draft
wl publish wechat ./article.html --mode publish --coverImage ./cover.png
wl publish wechat ./article.html --mode draft --autoCover
```

- `ai.image.defaultSize`

- `wechat.appSecret`
- `wechat.proxyOrigin`
